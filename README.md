# Trabajo2DIW
Trabajo 2º Trimestre D.I.W

# El tema de esta página es una tienda de electrónica de consumo, desde componentes a dispositivos electrónicos, como móviles, tablets, etc.

# Las funcionalidades de este sitio web serán las siguientes:

- Nos permite realizar login mediante un modal y registrarnos en el sitio.
- Nos permite realizar una búsqueda en la página desde el navbar.
- Nos permite consultar los mensajes recibidos desde el navbar.
- Nos permite visualizar productos mediante grid y carousel, visualizar un vídeo sobre las últimas novedades, así como filtrar los productos según marca, color y precio.
- Nos permite suscribirnos a una lista de difusión.
- Nos permite visualizar las redes sociales mediante tarjetas o "cards".
- Nos permite contactar con el vendedor desde el mismo footer.
